
# Point of Sale (POS) - Toko Berkah Utama

Aplikasi Kasir Pintar (Point of Sale) berbasis Android yang modern, cepat, dan mudah digunakan. Dibangun menggunakan teknologi terbaru Android Development untuk membantu manajemen toko retail atau UMKM.

## 🚀 Fitur Utama

- **Sistem Keamanan PIN**: Masuk ke aplikasi dengan PIN kasir yang aman.
- **Dashboard Interaktif**: Pantau Omzet harian, Laba bersih, dan Jumlah transaksi secara real-time.
- **Kasir Pintar**: Proses transaksi penjualan dengan pencarian produk cepat, manajemen keranjang, dan kalkulasi kembalian otomatis.
- **Manajemen Stok (Inventory)**: Kelola data barang, pantau sisa stok, dan peringatan otomatis jika stok menipis.
- **Data Pelanggan**: Simpan dan kelola database pelanggan untuk meningkatkan loyalitas.
- **Laporan Penjualan**: Laporan performa toko yang detail untuk membantu pengambilan keputusan bisnis.
- **Pengaturan Bisnis**: Kustomisasi nama toko, alamat, dan nomor telepon pada struk.

## 🛠️ Tech Stack

- **Bahasa**: [Kotlin](https://kotlinlang.org/)
- **UI Framework**: [Jetpack Compose](https://developer.android.com/jetpack/compose) (Modern Declarative UI)
- **Database**: [Room Persistence](https://developer.android.com/training/data-storage/room) (Local Data Storage)
- **Architecture**: MVVM (Model-View-ViewModel)
- **Dependency Management**: Gradle Version Catalog (libs.versions.toml)
- **Image Loading**: Coil
- **Local Storage**: DataStore / SharedPreferences

## 📱 Cara Menjalankan Aplikasi

1. **Clone Repositori**:
   ```bash
   git clone https://github.com/Cahyooo12/Pos-apps-kotlin.git
   ```
2. **Buka di Android Studio**: Pastikan Anda menggunakan versi terbaru (Ladybug atau lebih baru).
3. **Build & Run**: Hubungkan perangkat Android atau gunakan Emulator.
4. **Login Demo**:
   - Gunakan PIN: `1234` atau `0000` untuk masuk ke Dashboard.
